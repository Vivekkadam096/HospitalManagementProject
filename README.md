🏥 Vikhe Patil Hospital Management System (MERN Stack)

A full-stack hospital appointment booking system developed using the MERN stack (MongoDB, Express.js, React.js, Node.js).
The system allows patients to view doctors, book appointments, manage bookings, and generate QR-based appointment confirmations.

This project demonstrates modern web development, REST APIs, database management, and frontend–backend integration.

🚀 Features
👨‍⚕️ Patient Features

View hospital information

Check doctor availability

Book appointment online

Generate QR code for appointment verification

View appointments in dashboard

Edit appointment details

🏥 System Features

MERN stack architecture

REST API integration

MongoDB database storage

QR code appointment confirmation

Responsive UI design

Real-time appointment dashboard
